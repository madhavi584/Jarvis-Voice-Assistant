# Jarvis Voice Assistant

Jarvis Voice Assistant is a simple GUI-based assistant developed using Python. This project allows users to interact with an AI assistant through a graphical interface and text commands.

The assistant can perform basic tasks such as greeting the user, telling the current time and date, searching information on Wikipedia, and opening websites like Google and YouTube.

## Features
- GUI based interface
- Text to speech response
- Tell current time and date
- Wikipedia search
- Open websites like Google and YouTube

## Technologies Used
- Python
- Tkinter
- pyttsx3
- Wikipedia API

## How to Run

1. Install required libraries

pip install pyttsx3  
pip install wikipedia  

2. Run the program

python jarvis_gui.py
